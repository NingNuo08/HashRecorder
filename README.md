## HashRecorder 程序描述
### 概述
HashRecorder 是一个基于 Node.js 的文件哈希记录与完整性验证工具，提供中文用户界面，支持 Windows 平台。

### 核心功能
1. SHA256 哈希计算

- 计算单个文件的 SHA256 哈希值
- 支持大文件，显示计算进度
- 递归计算文件夹内所有文件的组合哈希
2. 完整性验证

- 自动检测文件是否被篡改
- 对比当前哈希值与历史记录
- 显示原始哈希与当前哈希的对比
3. 双文件夹对比

- 对比两个文件夹的差异
- 显示新增、删除、修改的文件
- 每个差异文件都显示对应的哈希值
4. 历史记录管理

- 自动保存哈希记录到本地文件
- 支持同一文件多次记录
- 验证时自动与最新记录对比
### 技术特点
- 纯 Node.js 实现 ：无需额外依赖
- 流式读取 ：支持大文件处理，内存占用低
- 中文界面 ：完整的中文用户界面
- 双模式启动 ：命令行 + 图形菜单
### 使用场景
- 文件完整性校验
- 检测文件是否被篡改
- 对比两个文件夹的差异
- 备份文件验证
